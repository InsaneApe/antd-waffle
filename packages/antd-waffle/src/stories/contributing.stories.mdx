# 贡献指南

这篇指南会指导你如何为 antd-waffle 贡献一份自己的力量，请在你要提 issue 或者 pull request 之前花几分钟来阅读一遍这篇指南。

## 请查看规范説明

- [提交信息规范](/docs/commit-lint)

## 开发流程

1. clone antd-waffle 的代码到本地

```
git clone https://github.com/FengBeans/antd-waffle
```

2. 安装项目依赖，推荐使用 yarn

```
yarn
```

3. 使用 yarn 安装每个 package 下的依赖

```
yarn install 

```

or 

```
npm run install 

```

4. 在本地运行 antd-waffle 的网站

```
yarn start
```

or 

```
npm run start
```
